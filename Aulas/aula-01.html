Aprendendo sobre as Tags principais e suas caracteristicas

Sobre as Tags:

Tags são elementos do HTML, com elas iremos criar diversas caixas com diferentes funções. Uma tag é composta de:

  • < - Sinal de menor
  • Nome da Tag
  • > - Sinal de maior
Isso é a estrutura básica de uma Tag. Algumas tags só terão essa estrutura, que chamamos de "abertura", mas 80% (ou mais) das Tags precisam de Aberta e Fechamento. O fechamento se dá pela seguinte estrutura:
  • < - Sinal de menor
  • / - Barra invertida
  • Nome da Tag
  • > - Sinal de maior
Juntando tudo isso, a nossa tag seria escrita da seguinte forma: <nomeDaTag> </nomeDaTag>
Tags como essa, escritas acima, precisam de um conteúdo, ou seja, um texto interno. Um exemplo é a tag <p>, sua estrutura é:
<p> Conteúdo do nosso Parágrafo </p>
Já tags como <img> são só isso, não precisam de conteúdo, então não tem fechamento.

Tags de estruturação do HTML:

  • <!DOCTYPE>: Mostra que estamos usando a última versão do HTML (v5)
  • <html>: Fala para o navegador que estamos criando uma estrutura HTML
  • <head>: É a cabeça da nossa página, todas as configurações que não vemos
  • <meta>: Configurações da página HTML
  • <title>: É o título do nosso HTML, é o texto que fica na aba do browser
  • <body>: É o corpo da nossa página, o que o usuário irá ver

Sobre atributos:

Atributos são como Caracteristicas de um Elemento HTML (tag). Imagine que o elemento é uma pessoa, e essa pessoa pode ter algumas caracteristicas, por exemplo:
<pessoa bonito="verdadeiro" genero="masculino" >

  • Pessoa é o elemento
  • Bonito e Genero são caracteristicas
  • Verdadeiro e Masculino são os valores dessas caracteisticas
