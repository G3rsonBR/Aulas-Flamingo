Aprendendo sobre as Tags principais e suas caracteristicas

  • Tag "DOCTYPE": Mostra que estamos usando a última versão do HTML (v5)
  • Tag "html": Fala para o navegador que estamos criando uma estrutura HTML
  • Tag: "head": É a cabeça da nossa página, todas as configurações que não vemos
  • Tag "lang": Define o idioma da nossa página
  • Tag "meta": Configurações da página HTML

      • Atributos do Meta:

    • charset: configuração dos caracteres doo HTML
    • name: Define qual o nome da configuração. Ex: viewport (define a janela de visualização)
    • content: Define o conteúdo da meta. Ex: "width=device-width, initial-scale=1.0" (Permite que a dela se adapte a todos os dispositivos)
  • Tag "title": É o título do nosso HTML, é o texto que fica na aba do browser
  • Tag "body": É o corpo da nossa página, o que o usuário irá ver
